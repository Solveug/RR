### Задача 47-1 — Сущности Хорошего Программиста 

Сайт хорошего программиста — Rails приложение. Основные объекты, которыми оперирует рельсовое приложение называются **моделями**, это сущности, которые обычно хранятся в базе данных и из свойств которых и формируется содержимое сайта.

Изучите внимательно сайт и составьте список **моделей** (сущностей, классов), из которых по-вашему он состоит.

Модели — это именно данные сайта. Классы, которые не содержат данных хранящихся в базе, типа `КалькуляторЦены`, угадывать не надо.
